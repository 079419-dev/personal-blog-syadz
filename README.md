# ✦ Irsyad Mustaqim — Personal Blog Portfolio

> A luxury black & gold themed personal blog and portfolio website built for CSD 34203 Special Topics in Software Development (CLO3).

---

## 📖 Description

A responsive personal blog and portfolio website showcasing my work as an IT student specializing in multimedia development at Universiti Sultan Zainal Abidin (UniSZA). The site features an elegant black and gold aesthetic with smooth animations, interactive elements, and a clean multi-page structure.

---

## ✨ Features

- 🏠 **Home Page** — Hero section with animated elements, skills ticker, expertise cards, and a featured quote
- 👤 **About Page** — Personal info, animated skill bars, education & experience timeline, and project showcase
- 📝 **Blog Page** — 3 blog posts with expandable full content on topics I'm passionate about (3D Design, Game Dev, Graphic Design)
- 📬 **Contact Page** — Contact information cards and a functional contact form with validation
- 🌙 **Black & Gold Theme** — Luxury aesthetic with animated background orbs, twinkling particles, and gold accents
- 📱 **Fully Responsive** — Mobile-friendly with hamburger navigation
- ✨ **Micro-interactions** — Gold particle effect on click, scroll fade-in animations, hover effects, pulsing availability indicator

---

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and semantic markup |
| CSS3 | Styling, animations, CSS variables, clip-paths |
| JavaScript (Vanilla) | Interactivity, scroll effects, IntersectionObserver |
| Google Fonts | Playfair Display, Cormorant Garamond, Rajdhani |
